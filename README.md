# MatchHub ⚽

> **matchhub.live** — Your complete FIFA World Cup 2026 hub. All matches, all info, all in one place. No popups. No piracy. No distractions.

## What is MatchHub?

MatchHub is the cleanest, most complete World Cup 2026 destination. Inspired by the structure of FIFA.com — built for fans who want **everything in one place** without the broken streaming experience.

## Three Products, One Mission

| Product | What It Does |
|---|---|
| 🌐 **Website (matchhub.live)** | Live scores, fixtures, highlights, news, players, standings, bracket — all in one beautiful PWA |
| 🧩 **Browser Extension** | Blocks popup ads, fake play buttons, and adult/gambling ads on streaming sites |
| 🤖 **Telegram Bot** | Daily updates, live goal alerts, and match reminders |

## Core Features

### 🎬 Match Experience
- 📅 **Full Schedule** — All 104 matches with real fixtures (e.g. *Mexico vs South Africa, Group A*)
- 🏳️ **Country Flags** — Every match card shows team flags
- 🔴 **Live Streaming Embed** — Official FIFA YouTube + country broadcaster embeds (legal)
- 🎥 **Highlights Hub** — All World Cup videos by year (2026, 2022, 2018, 2014...) — FIFA.com style
- 🏅 **Man of the Match** — Featured on every completed match card
- 📥 **Watch Replays** — Direct links to FIFA+ official replays (we don't host)

### 🎨 Beautiful Interactive UI
- ☀️ **Light mode default** + 🌙 dark mode toggle
- ✨ Smooth animations, transitions, micro-interactions
- 📱 Mobile-first, PWA installable
- ⚡ Sub-second loading even on 2G

### 📊 Data & Stats
- 🏆 **Group Standings** — Beautiful, live-updating
- 🪜 **Knockout Bracket** — Interactive, animated
- 👥 **Country Squads** — Full player rosters per team
- 📈 Player stats (goals, assists, cards)

### 📰 News & Updates
- 📰 **Authentic FIFA News** — Aggregated from FIFA.com, BBC Sport, ESPN (RSS)
- 🔔 **Smart Notifications** — Goals, reminders, match updates
- 🌐 **Multi-language** — English, Bangla, Urdu, Arabic, Hindi

## Tech Stack

- **Frontend:** Next.js 15 + Tailwind CSS + Framer Motion (animations)
- **Backend:** Next.js API routes + Supabase
- **APIs:** football-data.org (scores), YouTube Data API (highlights/live), FIFA RSS (news)
- **Extension:** Chrome Manifest V3
- **Bot:** Node.js + Telegraf
- **Hosting:** Vercel (web), Railway (bot), Chrome Web Store

## Why MatchHub

Current options are broken:
- **Pirate sites** = popups, malware, haram, illegal
- **ESPN/FIFA.com** = bloated, slow, English-only, paywalls
- **LiveSoccerTV** = links only, no real experience

**MatchHub gives you everything FIFA.com offers — faster, cleaner, free, multilingual, and 100% halal.**

## Legal & Ethics

- ✅ All live streams are **embedded official sources** (FIFA YouTube, broadcaster YouTube channels)
