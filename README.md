# FootballClean ⚽🛡️

> The cleanest way to watch the FIFA World Cup 2026 — no ads, no popups, no haram.

## What is FootballClean?

FootballClean solves the biggest problem football fans face: **finding a clean, distraction-free way to watch matches**. We don't host streams — we make legal streaming better.

## Three Products, One Mission

| Product | What It Does |
|---|---|
| 🌐 **Website** | Tells you where to watch every match legally (free + paid) in your country |
| 🧩 **Browser Extension** | Blocks popup ads, fake play buttons, and adult/gambling ads on streaming sites |
| 🤖 **Telegram Bot** | Daily updates on today's matches, live goal alerts, and reminders |

## Core Features

- 🔴 **Live Scores** — Real-time updates via free official APIs
- 📅 **Match Schedule** — All 104 World Cup 2026 matches in your timezone
- 🎬 **Highlights Hub** — Embedded FIFA official highlights after every match
- 📍 **Stream Finder** — Free legal streams by country
- 🕌 **Prayer Time Alerts** — Match conflict warnings for Muslim users
- 🏆 **Live Standings** — Group tables and knockout bracket
- 🔔 **Smart Notifications** — Goal alerts and match reminders
- 🌐 **Multi-language** — English, Bangla, Urdu, Arabic, Hindi

## Tech Stack

- **Frontend:** Next.js + Tailwind CSS (PWA-enabled)
- **Backend:** Next.js API routes + Supabase
- **APIs:** football-data.org (live scores), YouTube API (highlights)
- **Extension:** Chrome Manifest V3 + Vanilla JS
- **Bot:** Node.js + Telegraf
- **Hosting:** Vercel (web), Railway (bot), Chrome Web Store (extension)

## Why This Project

The football streaming experience is broken — full of popups, malware, and fake buttons. FootballClean fixes this **without piracy**: we point users to legal streams and clean up the experience around them. 100% halal, 100% legal, 100% useful.
