# MatchHub Brand Assets

## Files

| File | Use |
|---|---|
| `logo.svg` | Primary horizontal logo for light backgrounds (header, marketing) |
| `logo-dark.svg` | Horizontal logo for dark backgrounds (dark mode header) |
| `icon.svg` | Square icon (64x64) — use for favicon, PWA app icon, social avatar |

## Brand Colors

| Role | Hex | Where |
|---|---|---|
| Primary Green | `#22c55e` | Logo background, brand accent |
| Deep Green | `#15803d` | Gradient end on icon |
| Live Orange | `#f97316` | "Live" dot, `.live` text — energetic accent |
| Ink (Text) | `#0f172a` | Wordmark on light background |
| Snow (Text) | `#f8fafc` | Wordmark on dark background |
| Muted | `#64748b` / `#94a3b8` | Tagline (light/dark) |

## Brand Story

- 🟢 **Green** — football pitch, "go", live
- 🟠 **Orange dot** — live broadcast indicator, energy
- ⚽ **Soccer ball** — universal football icon
- **Tagline:** *"EVERY MATCH · EVERY MOMENT"*

## Usage

- Minimum width: 120px (logo), 32px (icon)
- Always preserve aspect ratio
- Don't recolor — use provided variants
